### Run Locally

To run **Tourly** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/starlabman/leader_tours.git
```

Windows:

```bash
git clone https://github.com/starlabman/leader_tours.git
```
